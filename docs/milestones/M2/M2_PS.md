# Weekly Product Pulse and Fee Explainer

Build an AI workflow that analyzes recent product reviews to generate a concise weekly product pulse and a structured explanation for a few common fee scenarios. The system clusters feedback into themes, extracts user quotes, and produces actionable insights.

Build an AI workflow for the same product you selected in Milestone 1 that:

- Converts a recent app review JSON into a weekly product pulse.  
- Generates a structured explanation for a few common fee/charge scenarios.   

The goal is to simulate how Product and Support teams use AI to generate structured internal updates and standardized explanations.

---

## What You Must Build

### Part A - Weekly Review Pulse

**Input:**
- 1 public reviews JSON (last 8–12 weeks)

**Your system must:**
- Group reviews into max 5 themes  
- Identify top 3 themes  
- Extract 3 real user quotes  
- Generate a ≤250-word weekly note  
- Add 3 action ideas  
- No PII in outputs.  

---

### Part B — Fee Explainer (Few Scenarios)

Pick a few fee scenarios relevant to your product  

*(e.g., exit load, brokerage fee, withdrawal charge, maintenance charge).*  

**Your system must:**
- Generate ≤6 bullet structured explanation for each scenario  
- Include 1 official source link for each scenario  
- Add: “Last checked: ”  
- Maintain neutral, facts-only tone  
- No recommendations or comparisons
